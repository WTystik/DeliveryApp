Старт проекта
-------------

1. Запуск бэкенда (.NET API)
---------------------------
Откройте PowerShell и выполните:

cd C:\Users\Roman\Desktop\DeliveryAppSolution\DeliveryApp\DeliveryApp

dotnet run

- Сервер будет доступен по адресу: https://localhost:5001
- Swagger: https://localhost:5001/swagger

2. Запуск фронтенда (React)
---------------------------
Откройте новую вкладку PowerShell и выполните:

cd C:\Users\Roman\Desktop\DeliveryAppSolution\frontend

npm install
npm start

- Откройте браузер: http://localhost:3000

----------------------

Как пользоваться
----------------
1. Создайте заказ через форму справа:
   - Выберите города отправителя и получателя из списка
   - Введите адрес в формате: ул. Пушкина д. 5 кв. 12 (автоподстановка шаблона)
   - Введите вес (от 0.01 до 1000 кг)
   - Выберите дату забора (год — только 4 цифры, дата должна быть в будущем)
   - Нажмите "Создать заказ"
2. Список заказов отображается слева
3. Детали заказа — кликните по заказу в списке
